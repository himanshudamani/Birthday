# Birthday Web Link (Mobile-First)

A clean, romantic birthday page built with plain HTML/CSS/JS (no frameworks).

## Add photos

1. Put your images in the `images/` folder (see `images/README.md` for exact filenames).
2. Paths are configured in `script.js` under `birthdayConfig.images` and each timeline entry's `image` field.
3. Missing images are hidden automatically — the page still works without them.

## Personalize in 2 minutes

1. Open `script.js`
2. Edit values in `birthdayConfig`:
   - `name`
   - `fromName`
   - `dateLabel`
   - `subtitle`
   - `letter`
   - `finalLine`
   - `timeline`
3. Save and open `index.html` in browser to preview.

## Host and share (recommended: GitHub Pages)

1. Create a new GitHub repository.
2. Upload these files (`index.html`, `styles.css`, `script.js`, `README.md`).
3. In GitHub repo go to **Settings -> Pages**.
4. Under **Build and deployment**:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Click **Save**.
6. Your link appears in 1-2 minutes:
   - `https://<your-github-username>.github.io/<repo-name>/`

Send that link to her.

## Why this is a good choice

- Free hosting
- Works perfectly for static romantic pages
- Fast and stable
- Easy to update anytime
