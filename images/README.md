# Add your photos here

Drop your images into this folder using these exact names:

| File | Where it appears |
|------|------------------|
| `cover.jpg` | Intro polaroid (floating photo at start) |
| `met.jpg` | Timeline — the day you met |
| `memories.jpg` | Timeline — everything in between |
| `proposal.jpg` | Timeline — proposal |
| `today.jpg` | Timeline — today |
| `letter-1.jpg` | Small photos inside the letter |
| `letter-2.jpg` | Small photos inside the letter |
| `gallery-1.jpg` … `gallery-5.jpg` | Film-strip memory scene (swipe + tap to enlarge) |
| `us.jpg` | Finale — heart-framed couple photo |

**Tips**
- Use `.jpg` or `.png` (update paths in `script.js` if you use `.png`)
- Square or portrait photos work best
- Keep each file under ~2 MB for fast loading on mobile

After adding photos, push to GitHub and the live site will update.
